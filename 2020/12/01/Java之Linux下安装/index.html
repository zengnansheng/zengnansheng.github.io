Java之Linux下安装

环境：

Centos7.6

下载linux版Jdk,比如jdk-xxx.tar.gz

用shell客户端，比如xshell登录到linux终端，用户名用root

将jdk文件通过ftp软件传到linux下/usr/local/software，比如xftp

解压jdk文件包,并重命名为jdk

cd /usr/local/software

tar -zxvf jdk-xxx.tar.gz

mv jdk-xxx jdk

卸载centos原本自带的openjdk，运行命令：rpm -qa | grep java

然后通过 rpm -e –nodeps 后面跟系统自带的jdk名 这个命令来删除系统自带的jdk，

例如：rpm -e –nodeps java-1.8.0-openjdk-1.8.0.102-4.b14.el7.x86_64

编辑配置文件，配置环境变量

vim /etc/profile

末尾添加如下内容：

export JAVA_HOME=/usr/local/software/jdk

export CLASSPATH=.:$JAVA_HOME/jre/lib/rt.jar:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar

export PATH=$PATH:$JAVA_HOME/bin

设置执行环境变量的配置生效

source /etc/profile

查看安装情况

java -version
